새 로봇 탭이 추가되지 않았습니다

  • 모든 장비의 ROS_DOMAIN_ID가 같고 ROS_LOCALHOST_ONLY=0인지 확인하세요.
  • 같은 Wi-Fi 또는 유선망에 연결하고, 공유기의 클라이언트 격리와 PC 방화벽의 DDS 멀티캐스트 차단을 해제하세요.
  • 각 로봇에서 base bringup을 실행한 뒤 /scan, /odom, /cmd_vel 중 하나 이상이 발행되는지 확인하세요.
  • 토픽이 루트(/scan)인지 네임스페이스(/tb3_3/scan)인지 확인하세요.
  • SSH 응답 장비는 로봇으로 단정하지 않습니다. ROS 토픽이 확인돼야 새 탭으로 등록됩니다.
